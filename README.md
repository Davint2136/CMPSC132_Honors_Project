# CMPSC132_Honors_Project

HOW TO RUN:

1. Put your list of distribution sizes in the DIST_RUN_SIZES variables in the main functions of both data_analysis.py and data_gen.py

2. Set the number of runs per algorithm in RUNS_PER_ALG in data_gen.py

3. Change your working directory to the same as this repository

4. Run data_gen.py

5. Run data_analysis.py to get the graph.
